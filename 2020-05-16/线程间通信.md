# 线程间通信

| 通信类型 | 关键字                                                       | 详细描述                                   |
| -------- | ------------------------------------------------------------ | ------------------------------------------ |
| 共享内存 | ``volatile``                                                 | 内存屏障保证数据安全<br />多线程间保证可见 |
| 状态传递 | ``wait/notiry/notifyAll``<br />``join``                      | 共享内存条件检测<br />动态变更线程状态     |
| 管道传输 | ``PipedReader/PipedWriter``<br />``PipedInputStream/PipedOutputStream`` | 管道数据直接传输                           |
