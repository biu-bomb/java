# 为什么Redis这么快

| 手段         | 影响                                                        |
| ------------ | ----------------------------------------------------------- |
| 操作基于内存 | 内存操作，十分快速                                          |
| 数据结构简单 | 专门设计的数据结构，简单高效                                |
| 单线程任务   | 单线程，摈弃上下文切换和竞争条件，无死锁问题和加锁开销      |
| IO多路复用   | 非阻塞IO多路复用                                            |
| Reids-VM     | 1. 冷热数据分离<br />2. 更小的压缩体积<br />3. 无需阻塞线程 |
